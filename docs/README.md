# 🧸 Antigravity SDK Python (ELI5 - Einfach erklärt für 5-Jährige)

## Was macht dieses Projekt?
Stell dir vor, du hast viele schlaue Spielzeug-Roboter (wir nennen sie **Agenten**). Diese Roboter wollen alle zusammen ein großes Lego-Schloss bauen. Damit sie sich nicht im Weg stehen und miteinander reden können, brauchen sie Hilfe.

Dieses Projekt ist wie ein **magisches, unsichtbares Straßennetz** (ein *Mesh-Netzwerk*):
1. **Wer macht was?** Es malt eine Karte, auf der genau steht, welcher Roboter mit welchem Werkzeug befreundet ist (das ist der *Abhängigkeitsgraph*).
2. **Keine Unfälle:** Es passt auf, dass nicht zwei Roboter gleichzeitig durch dieselbe kleine Tür gehen wollen (das nennen wir *Port-Leasing*).
3. **Schlaue Gedanken teilen:** Wenn ein Roboter eine tolle Idee hat, flüstert dieses System sie superschnell allen anderen Robotern ins Ohr (das ist die *Vektorsuche* und *WebAssembly-Ausführung*).

## Warum existiert es?
Damit unsere schlauen Roboter perfekt als Team zusammenarbeiten können, ohne sich zu streiten oder gegeneinanderzulaufen!
